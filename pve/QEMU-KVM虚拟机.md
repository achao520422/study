# QEMU (short form for Quick Emulator)
```text
QEMU（Quick Emulator 的简称）是一个开源虚拟机管理程序，它模拟 物理计算机。从 QEMU 所在的主机系统的角度来看 运行,QEMU 是一个用户程序，可以访问许多本地资源 like 分区、文件、网卡，然后传递给 模拟计算机，将它们视为真实设备。

在模拟计算机中运行的客户机作系统会访问这些 设备运行，就像在真实硬件上运行一样运行。例如，您可以将 作为 QEMU 参数的 ISO 映像，以及在模拟计算机中运行的作系统 将看到一个真正的 CD-ROM 插入 CD 驱动器。

QEMU 可以模拟从 ARM 到 Sparc 的各种硬件，但 Proxmox VE 是 只关心 32 位和 64 位 PC 克隆仿真，因为它表示 绝大多数服务器硬件。PC 克隆的仿真也是其中之一 由于处理器扩展的可用性，速度最快，这大大 当模拟架构与主机相同时加速 QEMU 建筑。
```
